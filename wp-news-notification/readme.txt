New Post Push Notification for Android App.

== Description ==

Wordpress Plugin for Automated and custom push notification to android device by using FCM Server API.

= Docs & Support =

You can visit github page for detail documentation and support : https://github.com/SagarMaher2/wp-news-notification

= Privacy Notices =

With the default configuration, this plugin, in itself, does not:

* track users by stealth;
* write any user personal data to the database;
* send any data to external servers;

== Installation ==

1. Upload the entire `wp-news-notification` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the WPNews Push Notification->WPNews Settings screen to configure the plugin

You will find 'WPNews Push Notification' menu in your WordPress admin panel.

For basic usage, you can also have a look at the [plugin web site](https://github.com/SagarMaher2/wp-news-notification).
